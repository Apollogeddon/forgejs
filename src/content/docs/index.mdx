## Project Overview

Forge.js centralises configuration management for Node.js projects. It consolidates build, test, linting, and release settings into a single installable package, reducing the need for maintenance of individual dotfiles across repositories.

<CardGrid>
  <Card title="Standardised Tooling" icon="setting">
    Provides default configurations for **Biome**, **Vitest**, and **Tsup** to enforce consistency across multiple projects.
  </Card>
  <Card title="Reusable Workflows" icon="rocket">
    Exports modular **GitHub Actions** workflows for testing, building, and releasing packages, eliminating manual YAML authoring.
  </Card>
  <Card title="Project Scaffolding" icon="puzzle">
    Includes a CLI tool to bootstrap new repositories with the `init` command. Supports modular setup for libraries, Debian packages, and more.
    ```bash
    npx @apollogeddon/forgejs init [options]
    ```
  </Card>
  <Card title="Automated Releases" icon="list-format">
    Implements **Semantic Release** pipelines for automated versioning, changelog generation, and package publishing.
  </Card>
</CardGrid>

## Tech Stack

The project utilises specific tools chosen for performance and ESM compatibility.

<CardGrid>
  <Card title="Biome" icon="biome">
    High-performance linting and formatting (Rust-based). Replaces ESLint and Prettier.
  </Card>
  <Card title="Tsup" icon="node">
    Zero-config TypeScript bundler powered by esbuild.
  </Card>
  <Card title="Vitest" icon="seti:vite">
    Vite-native unit test framework with ESM support and coverage reporting.
  </Card>
  <Card title="Semantic Release" icon="rocket">
    Automated version management based on conventional commit history.
  </Card>
  <Card title="Lefthook" icon="laptop">
    Git hooks manager for enforcing quality checks locally.
  </Card>
  <Card title="Commitlint" icon="seti:git">
    Validator for Conventional Commits specification adherence.
  </Card>
  <Card title="Publint" icon="magnifier">
    Validator for `package.json` exports and compatibility.
  </Card>
  <Card title="Snodeb" icon="linux">
    Debian package generator for Node.js applications.
  </Card>
</CardGrid>
